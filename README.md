# basic_clock
Welcome to Basic Clock. Basic clock is project that I have been working on for many years. It started off as Corona cross platform solution, then it progressed to native iOS Objective C solution. In it's lattest incarnation I've rebuilt it for a third time from the ground up in Flutter.

Basic clock now has the smooth animation it had in its origional Corona solution. Both the second and the minute hand transition smoothly.

You are welcome to review the code here or report an bugs.
