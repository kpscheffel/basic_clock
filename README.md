# basic_clock
Basic Clock is exactly what it says. It's a basic analogue clock.

Note the updated version of the iOS clock is currently available on the Apple App Store: https://apps.apple.com/za/app/basic-clock/id421899094. Google Play Store version is currently not available but should be up again soon after Google requested a change to the description. When it is back you will be able to install it at : https://play.google.com/store/apps/details?id=za.co.scheffel.basic_clock 

Welcome to Basic Clock. Basic clock is a project that I have been working on for many years. It started off on a mobile cross-platform solution back in 2011, then it progressed to native iOS Objective C solution. In its latest incarnation, I've rebuilt it for a third time from the ground up in Flutter.

Basic Clock now has the smooth animation it had in its original cross-platform solution. Both the second and minute hand transition smoothly.

You are welcome to review the code here or report an bugs.
