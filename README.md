# basic_clock
Basic Clock is exactly what it says. It's a basic analogue clock.

Note the updated version of the iOS clock will be available by mid January 2020. Current is not recommended to be downloaded. Google Play Store version is up and running, install now : https://play.google.com/store/apps/details?id=za.co.scheffel.basic_clock 

Welcome to Basic Clock. Basic clock is a project that I have been working on for many years. It started off as Corona cross-platform solution, then it progressed to native iOS Objective C solution. In its latest incarnation, I've rebuilt it for a third time from the ground up in Flutter.

Basic Clock now has the smooth animation it had in its original Corona solution. Both the second and minute hand transition smoothly.

You are welcome to review the code here or report an bugs.
